# History of Philosophy

An interactive visual map tracing philosophical thought from ancient foundations to modern times.

## View

Open `index.html` in a browser, or visit the [live site](https://tylerm2230.github.io/history-of-philosophy/).

## Navigate

- **Pan** — click and drag
- **Zoom** — scroll wheel or pinch
- **Search** — find philosophers by name
- **Click nodes** — view details

## Data

Philosophy.canvas contains the source data in Obsidian canvas format.

## Deploy

Push to GitHub and enable Pages in repository settings.
